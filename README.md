# Sakina (سكينة)

*Making the invisible visible — the neural landscape of Quran recitation.*

Sakina visualizes how the human brain responds to Quran recitation, rendering the neural experience as Islamic geometric patterns in real-time.

## How it works

1. **Listen** — Play any surah from Quran.com's recitation library
2. **Predict** — TRIBE v2 (Meta) predicts the brain's neural response to the audio
3. **Visualize** — tenfold renders Islamic geometric patterns reflecting the brain state

Peace becomes geometry. Awe becomes fractals. The divine word, made visible.

## Tech

- [Quran.com API](https://api.quran.com) — audio recitations + verse text
- [TRIBE v2](https://github.com/facebookresearch/tribev2) — neural response prediction
- [tenfold](https://github.com/cookmom/tenfold) — Islamic geometric pattern engine
